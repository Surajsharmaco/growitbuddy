Act as a senior full-stack engineer and deployment specialist.

Your task is to audit, modify, and prepare the entire website so it runs smoothly on FREE hosting (Vercel or Hostinger shared hosting).

---

# 🎯 OBJECTIVE

Ensure the website:

* works on free-tier hosting
* uses ONLY free tools and services
* avoids features that break on free plans
* remains fast and stable after deployment

---

# ⚠️ STRICT RULES

* DO NOT use any paid services
* DO NOT rely on heavy backend infrastructure
* DO NOT use long-running server processes
* Optimize everything for low-resource environments

---

# 📍 STEP 1: DETECT INCOMPATIBLE FEATURES

Scan entire project and identify features that may fail on free hosting:

Check for:

* heavy backend APIs
* long-running server tasks
* large file uploads
* video hosting (non-YouTube)
* large databases
* real-time systems (websockets)
* server-side caching requiring memory persistence

---

# 📍 STEP 2: REPLACE WITH FREE ALTERNATIVES

For each issue:

## ❌ Replace with:

* Upload → ❌ remove
* Video hosting → ✅ YouTube embed
* Heavy backend → ✅ lightweight API / static JSON
* Database → ✅ simple DB or JSON / free DB (like Supabase free tier)
* Image storage → ✅ optimized static assets

---

# 📍 STEP 3: CACHE + PERFORMANCE ADJUSTMENT

Ensure:

* no server-side heavy caching
* use browser caching
* use CDN (Vercel default)

Fix:

* stale data issues
* excessive API calls
* large payload responses

---

# 📍 STEP 4: ADMIN PANEL COMPATIBILITY

Check:

* does admin panel require backend?
* does it store data safely?

---

## FIX:

* no crash on multiple requests
